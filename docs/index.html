EventGen

EventGen e una base di sviluppo per costruire un'app desktop completa: selezione area, generazione fonti compatibili e analisi eventi in un flusso unico.

Apri il progetto Quickstart Formato sources.yaml Configurazione
1. Configura Definisci area, categorie e fonti in file YAML esterni.
2. Esegui Lancia la pipeline manualmente con python3 run.py.
3. Pubblica Ottieni un events.html statico pronto da aprire o condividere.

Cosa include

  • fetch di fonti HTML, WordPress e pagine dinamiche opzionali;
  • parsing euristico con supporto IA opzionale;
  • deduplica per titolo, data e comune;
  • affidabilità alta, media, bassa;
  • output HTML con filtri, ricerca e ordinamento.

Regole chiare

  • i social servono solo a scoprire eventi;
  • gli eventi senza data reale vengono esclusi;
  • news, editoriali e attività permanenti non entrano;
  • la logica non è hardcodata su un singolo territorio.

Uso minimo

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 run.py

Per chi è utile

Comuni, proloco, reti territoriali, musei, redazioni locali, operatori turistici, community manager e sviluppatori che vogliono un punto di partenza serio ma leggero.

Progetto pensato per essere riutilizzabile, documentato e pubblicabile anche tramite GitHub Pages.
